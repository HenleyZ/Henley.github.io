Projects

  • Arcadia - Unreal Engine (Click for demo)

    This is my favorite project. It is essentialy Valorant replaced with anime chanracters built from the ground up in Unreal Engine with a server-authoritative design. The game has an anime like art style thanks to a custom shader source build. One of the key features is client-server replication to prevent modified client hacks. The hardest part was developing a custom server-validated line trace lag compensation system to mitigate the effects of ping. Otherwise, other players would have to end up leading their shots based on how laggy they are, which goes again the design of precise gun play. I emulated keys aspects of Valorant/CSGO gameplay (weapons, spray pattern, recoil, movement, projectiles, bullet penetration). This game is still heavily being working on, and I am at the point of a working multiplayer demo over steam. Unlike most FPS development projects in unreal, I didn't not use anny weapon system asset packs.

  • Study Buddy

    An online platform that collects information about libraries across the UC Berkeley campus and recommends the user a selection of choices based on user preference input. Accessible information about libraries which the “filter by” feature will be based upon includes a live crowd meter, user reviews, open hours, and more. Built With: React.js, Axios, Chakra UI, MongoDB, Express.js, Node.js. Features

    • Algorithm thats generate an ordered list that tries to align with user inputed preferences
    • Dynamic router for procedural website per library
    • API to gather live/estimated busyness at a library
    • Caching system to limit API calls
    • Authentication and login system to save user preferences
    • Review and rating system per library
    • Libraries are not hard coded, easy to add additional places.
